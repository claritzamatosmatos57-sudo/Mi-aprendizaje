Título del proyecto:

Tutorial interactivo: Cómo abrir una puerta

Descripción del proyecto:

> Este es mi primer proyecto en HTML. En él practiqué el uso de etiquetas básicas como <h1> a <h3>, <p>, <div> con estilos inline, y enlaces <a>.

El proyecto muestra un pequeño tutorial creativo sobre cómo abrir una puerta, usando colores y secciones para organizar el contenido.

Con este ejercicio aprendí a estructurar información en una página web, probar el código en el navegador y corregir errores de sintaxis.



Qué aprendí:

Estructura básica de HTML (<!DOCTYPE html>, <html>, <body>)

Encabezados y párrafos (<h1>, <h2>, <h3>, <p>)

Divisiones y secciones (<div>)

Estilos inline (style="") para colores y padding.
